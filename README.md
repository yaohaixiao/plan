# Plan

Plan 是一个纯 JavaScript 开发的 Todo 类型的任务管理应用。使用 Plan，无需注册，不用绑定账号，打开浏览器就可以使用，让你只用专注当前的工作。

## 功能特点

- 支持任务的新增、编辑、移除；
- 任务份 4 个等级，分别用桃、心、梅、方表示；
- 支持使用本地缓存持久保存任务；
- 支持拖拽任务改变状态，操作简单方便；
- 支持重要任务标记；
- 界面美观大方，有 8 个主题可用选择；
- 任务描述支持 markdown 语法；
- 响应式布局，方便移动设备访问，且针对移动设备做了更加人性化的操作交互；
- 原生 JavaScript 开发，性能绝对有保障；

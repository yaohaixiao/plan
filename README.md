# plan

Plan 是一个纯 JavaScript 开发的 Todo 类型的任务管理应用。开始是为我的一个朋友教学生如何管理好自己时间用的一个教学演示模型。不过，我想应该也有很多朋友不想把什么都存在云上，只希望有个简单的工具管理最近的任务，所以选择开源了 Plan。 没有注册，没有号码绑定，你只要专注管理只属于你的任务就是了。

## 功能
- 支持设置任务等级（桃、心、梅、方）；
- 支持标志重要任务；
- 任务描述支持 markdown 语法；
- 支持移动设备访问；
- 界面简洁，操作简单；

## Dependencies
